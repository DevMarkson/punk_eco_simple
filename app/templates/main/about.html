À propos de Punk Eco

Mission

Punk Eco est un tableau de bord économique intelligent dédié au suivi et à l'analyse des indicateurs économiques marocains. Notre mission est de fournir une vue d'ensemble claire et accessible de l'économie nationale.

Fonctionnalités

  • Tableau de bord interactif avec les indicateurs économiques clés
  • Authentification utilisateur avec gestion des rôles
  • API REST pour l'accès aux données économiques
  • Visualisations avancées avec Plotly/Dash
  • Export de données en CSV, Excel et JSON
  • Recherche avancée et filtrage des indicateurs
  • Alertes pour les variations significatives d'indicateurs

Technologies

Cette application est construite avec :

  • Flask (Python) - Framework web
  • SQLAlchemy - ORM pour la base de données
  • Plotly/Dash - Visualisations interactives
  • Bootstrap - Interface utilisateur
  • SQLite/PostgreSQL - Base de données

Version

Version actuelle : 0.1.0
